# MonitoringTool_v2

Anforderungen: psutil, icmplib

Funktionen:
- Auslesen einer Config Datei
- Logging
- Mail
- Auslesen von Systemparametern (CPU-Usage, RAM-Usage, Disk-Usage, Network)
- Ausgeben der Werte

Alle Funktionen getestet in Windows 11 und Debian 11
